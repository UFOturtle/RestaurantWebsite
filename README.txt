This project is a prototype for a cafe website which uses nodejs, express, axios, react, bcrypt, json webtoken
and other middleware to manage a menu where customers will be able to order carryout and the owners can recieve
notifications of orders via text using twilio or email using mailgun.
Another possible outcome is to integrate the backend with a ticket system to print order slips

Tasks:
-Add in shopping cart feature using express sessions
-Build out home page
-Add buttons to build shopping cart
-Add route for ordering menu items
-Add email, sms notifications
-Use SAM (Serverless Application Model) to make site Serverless
-Make authentication tokens expire
-Add review system to menu items

**NOTE**
Tasks are high level and will be further elaborated upon once they started
